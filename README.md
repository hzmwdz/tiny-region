# TinyRegion

## 安装

```bash
composer require hzmwdz/tiny-region
```

## 发布语言

```bash
php artisan vendor:publish --tag="tiny-region-lang"
```

## 发布迁移

```bash
php artisan vendor:publish --tag="tiny-region-migrations"
```

## 导入

```bash
php artisan vendor:publish --tag="tiny-region-sql"
```
